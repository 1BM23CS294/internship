# Intelligent Resume Analyzer

**Industry:** Human Resources Technology

### Skills Gained
- Python Programming
- Text Processing
- Data Extraction Algorithms
- JSON File Handling
- Matching Algorithms

---

### Description
Build a Python-based application that automates resume screening by parsing resumes, matching candidates to job requirements, and generating detailed analysis reports. The system should extract key information from resumes, calculate match scores, and provide hiring recommendations.

---

### Evaluation Criteria
- **Code Quality:** Clean, well-structured Python code following PEP 8 guidelines.
- **Parsing Accuracy:** Correct extraction of name, email, skills, and experience from resumes.
- **Matching Algorithm:** Accurate calculation of match scores (0-100) based on job requirements.
- **File Operations:** Proper saving and loading of data in JSON format.
- **Report Generation:** Clear, professional analysis reports with recommendations.
- **Error Handling:** Graceful handling of edge cases and invalid inputs.

---

### Outcome
A complete resume analysis system that automates candidate screening, provides match scores, and generates detailed hiring reports for recruiters.
